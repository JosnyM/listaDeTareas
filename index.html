Lista de Tareas

Bienvenido/a a la aplicación web "Lista de tareas"

- Nuestra aplicación te permitirá organizar tus tareas, asignar fechas y prioridad

- Una vez completada la tarea podrás tacharla

- También podrás limpiar la lista de tareas y borrarlas

- Quedaran almacenadas en caso de que necesites acceder luego
